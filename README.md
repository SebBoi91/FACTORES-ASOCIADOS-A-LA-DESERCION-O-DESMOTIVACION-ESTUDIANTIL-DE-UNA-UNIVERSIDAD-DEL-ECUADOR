# FACTORES-ASOCIADOS-A-LA-DESERCION-O-DESMOTIVACION-ESTUDIANTIL-DE-UNA-UNIVERSIDAD-DEL-ECUADOR
Este estudio identifica los factores asociados a la deserción y desmotivación estudiantil en una universidad ecuatoriana mediante un enfoque cuantitativo y correlacional. Se aplicó un cuestionario validado a 300 estudiantes, analizando dimensiones como motivación académica, estrés, satisfacción con la carrera y apoyo institucional. Mediante análisis estadísticos descriptivos, correlacionales y factorial exploratoria (AFE), se identificaron tres factores latentes que explican la intención de abandono. Los resultados destacan que el estrés académico, la baja motivación y la percepción de bajo apoyo institucional son predictores significativos de la deserción. Se proponen estrategias institucionales basadas en evidencia para mejorar la retención estudiantil.
